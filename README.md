# Predictive-Analytics-for-Forex-Market-Behavior
Built a forex market USD/EUR prediction system using Statistical Model (AR, MA, ARMA, ARIMA, SARIMA, SARIMAX),Point Probabilistic Model(Quantile Regression, Conformal Prediction), Tree-based-ML Models(RF, XGBoost, LightGBM), Deep-Learning Models(RNN, LSTM, GRU, TCN, MLP, and N-BEATS), with help of time-series analysis and forecasting techniques.
